File transfer unit WEB
Данный проект сделан Глазков Вячеслав и Шагеев Тимур.

В этом приложении можно "перекинуть" файл без потери качества вашему другу. Для этого друг, который передает файл, закодит на 
страничку "Передать", загружает файл и передает код своему другу. Друг вводит этот код на страничке "Получить" и кайфует.
